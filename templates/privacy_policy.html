# Politique de Confidentialité ## Introduction Nous respectons votre vie privée et nous nous engageons à protéger vos données personnelles. Cette politique de confidentialité vous informe sur la manière dont nous collectons, utilisons, et protégeons vos données lorsque vous utilisez notre site web. ## Données Collectées Nous pouvons collecter les types de données personnelles suivants : - Informations de contact (nom, adresse e-mail). - Données de navigation (adresse IP, type de navigateur, pages visitées). - Informations fournies volontairement par l'utilisateur (messages envoyés via le formulaire de contact). ## Méthodes de Collecte Nous collectons les données personnelles via : - Formulaires en ligne. - Cookies et technologies similaires. ## Utilisation des Données Nous utilisons vos données pour les finalités suivantes : - Fournir et améliorer nos services. - Répondre à vos demandes et communiquer avec vous. - Analyser l'utilisation de notre site web pour en améliorer la performance. ## Protection des Données Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, modification, divulgation ou destruction. ## Partage des Données Nous ne partageons pas vos données personnelles avec des tiers, sauf dans les cas suivants : - Fournisseurs de services tiers qui nous assistent dans l'exploitation de notre site web. - Lorsque la loi l'exige. ## Vos Droits Vous disposez des droits suivants concernant vos données personnelles : - Accès à vos données. - Rectification de données incorrectes. - Suppression de vos données. - Opposition à l'utilisation de vos données. ## Contact Pour toute question ou demande concernant vos données personnelles, vous pouvez nous contacter à : [votre adresse e-mail de contact]. ## Modifications de la Politique Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page. Date de dernière mise à jour : [date].
